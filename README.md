# PaperEdit-support
Create Edit Wall for Film and Tv from pdf
Getting Started

Launch PaperEdit

Choose File → Import Script PDF…

Select a script PDF

Scenes will be created automatically

Select a scene tile and click Import to attach still images

Use File → Print… to print the wall layout

Image Import

Batch image import matches filenames by scene number at the start of the filename:

Example:
23A_kitchen.png → attaches to Scene 23A

Troubleshooting

If images do not attach during batch import, ensure the scene number appears first in the filename.
